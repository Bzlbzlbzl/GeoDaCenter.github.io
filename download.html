GeoDa 1.8+ Beta Release

This 1.8 beta release is the most current version of GeoDa with new features. It is a beta release since we have only tested the program internally. We did find it to be stable but if you encounter a bug, please let us know. You can also report bugs and issues on our Github site. This version is continuously updated when bugs are fixed.

Latest 1.8 beta release: 6/11/2016 subversion 1.8.8 (release notes)

  • GeoDa for Windows (XP, Vista, 7, 8, 8.1 and 10) beta version 1.8.8 (32-bit and 64-bit)
  • GeoDa for OS X (10.6.8 or later) beta version 1.8.8
  • GeoDa for Linux (Ubuntu 14.04 or later) beta version 1.8.6

The source code for GeoDa is available here and is released under the GPL v3.0 license.
