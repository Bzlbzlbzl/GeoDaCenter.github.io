GeoDa 1.8+ Beta Release

This 1.8 beta release is the most current version of GeoDa with new features. It is a beta release since we have only tested the program internally. We did find it to be stable but if you encounter a bug, please let us know. You can also report bugs and issues on our Github site. This version is continuously updated when bugs are fixed.The source code for GeoDa is available here and is released under the GPL v3.0 license.
